CSS Positioning

CSS Positioning: Overview

• position defines alternative positioning rules for elements

• Useful for advanced layouts beyond float and flex

• Enables use of top, right, bottom, left

• position has four values:


	position: static;
	position: fixed;
	position: relative;
	position: absolute;
position: static;

• Default CSS positioning value for ALL elements

• Static elements are NOT positioned

• Cannot set the following properties:

⇒ top

⇒ right

⇒ bottom

⇒ left

position: fixed;

• Positioned relative to the browser window

• Element will stay in view, even as window scrolls

• Removed from the normal flow of the document

• Used for for persistent navigation: "sticky headers"

• Can position using via directional CSS

Sticky Header Example

• Apply the following CSS rules to your <header>:


 header {
   position: fixed;
   width: 100%;
   top: 0;
   height: 50px; // should match the padding-top on body
 }

• Apply the following CSS rules to your <body>:


 body {
   padding-top: 50px; // should match the height of header
 }

CSS Transitions

CSS Transitions

• Provide a method for controlling CSS changes

• Ex: Slowly fade color change over a second

• transition: property duration function delay;

• Applied:


	p {
	  transition: color 2s;
	}
CSS Transitions

Codealong: Animated Sticky Navigation Bar

1) Open animated-sticky-nav

2) Apply fixed positioning and transition: all 2s; to the <header>

4) Create an "offset" class:


	.header-offset {
	  background: red;
	  font-size: 30px;
	}

5) Use jQuery to listen for scrolling and toggle the CSS class


	$(window).scroll(function () {
	  var offset = $(window).scrollTop();

	  if (offset > 150) {
	    $('header').addClass('header-offset')
	  } else {
	    $('header').removeClass('header-offset')
	  }
	})
position: relative;

How would we create this?

position: relative;

• Positioned relative to its normal position

• Move using directional CSS properties

• Note: the original space for the element is preserved

• Used for:

⇒ Overlapping elements

⇒ As a reference for absolutely-positioned elements

• CodePen exercise

position: absolute;

• Positioned relative to either:

⇒ Browser window

⇒ Closest positioned element

• Removed from the normal flow of the document

• Used for:

⇒ Precisely placing an element within a parent

⇒ Animations

• CodePen Exercise

absolute & relative

• Use position: absolute with position: relative to absolutely position an element within another element

• position: relative is placed on the parent element

• position: absolute is placed on the child element
