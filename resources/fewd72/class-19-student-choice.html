<meta> Tags

<meta> Tags

• Metadata is data about data...

• The <meta> tag provides data about your HTML

• Contains name and content attributes

• Ex: <author>:


	<meta name="author" content="Keanu Reeves">

<meta> Tags

• <description>:


<meta name="description" content="This will show up in SERP's!!">

• <keywords>:


<meta name="keywords" content="Google, doesn't, care, about, this">

• <viewport>:


<meta name="viewport" content="width=device-width, initial-scale=1">

<meta> Tags

Notes on Search Engine Optimization:

• As of 2009, Google no longer considers the keywords <meta> tag

• description and <title> will appear in SERP's

• Check out rich snippets

Minify JS and CSS

Minifiy JS and CSS

Minification:

...in computer programming languages and especially JavaScript, is the process of removing all unnecessary characters from source code without changing its functionality.

Source: Wikipedia

Minifiy JS and CSS

• Unnecessary characters:

⇒ white space (spaces and tabs)

⇒ new line

⇒ comments

• Help with readability; are not required to execute:


	'cat' + 'dog'

	// is the same as:

	'cat'+'dog'

Minifiy JS and CSS

Why?

• JS/CSS files are trasmitted over the internet

• The larger the file-size, the more time it will take to download

• Minification reduces overall file-size:

⇒ jQuery = 258 KB

⇒ jQuery Minified = 86 KB

⇒ Roughly 2/3 smaller!

⇒ http://jscompress.com/
