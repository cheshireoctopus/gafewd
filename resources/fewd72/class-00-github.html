git/github

git - What is it?

git ≠ GitHub

git is a version control program

Takes a snapshot of your project's current state

Allows developers to work on projects simultaneously

git - some vocab words

REPOSITORY: a central location in which data - typically about a project - is stored and managed

CLONE: download data (a repository) from the cloud to your local machine (laptop, computer, etc.)

COMMIT: takes a snapshot of your local code

PUSH: Pushes a commit to the repository

GitHub - What Is It?

GitHub is a platform makes it easy to work with git

Like Dropbox, but for code

Stores a history of files and their changes

Hosts files in the cloud; easy to share finished product

GitHub - Why Do Developers Use It?

GitHub stores history; we can easily fix mistakes

GitHub allows multiple developers to work on the same project

Tracks change so you can see who worked on (or, broke) what

Allows for line-by-line feedback

GitHub - How Will We Use It?

1. All course materials/homework will be posted on our GitHub organization

2. You will clone the course material/homework repos

3. Build the assignments in your repo

4. Push your finished assignments to GitHub

5. We will review/comment your code
